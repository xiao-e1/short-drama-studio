本地启动
只需三步

  1. 准备环境：Node.js 18+ 与 ffmpeg。
  2. 安装依赖：进入 backend 执行 npm install。
  3. 打开应用：首次访问时输入自己的 Agnes API Key。
查看完整 README
$ git clone <repository-url>
$ cd short-drama-studio/backend
$ npm install
$ npm start
